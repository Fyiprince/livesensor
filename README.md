# Air Pressure System (APS) Failure Detection

## Project Overview

This project focuses on detecting failures in the Air Pressure System (APS) of heavy-duty vehicles using Machine Learning.

The Air Pressure System is a critical component responsible for vehicle braking. It uses compressed air to force a piston that applies pressure to the brake pads, helping the vehicle slow down or stop safely.

The objective of this project is to predict whether a vehicle failure is caused by a component of the APS or by another component in the vehicle.

---

## Problem Statement

The Air Pressure System (APS) is a critical component of a heavy-duty vehicle that uses compressed air to force a piston, providing pressure to the brake pads and slowing the vehicle down.

The benefits of using APS instead of a hydraulic system include:

- Easy availability of natural air
- Long-term sustainability
- Reduced maintenance requirements
- Suitability for heavy-duty applications

This is a Binary Classification problem where:

- Positive Class (1): Failure caused by a component of the APS.
- Negative Class (0): Failure caused by components other than the APS.

The goal is to develop a predictive model capable of identifying APS-related failures accurately.

---

## Dataset Description

The dataset contains sensor measurements collected from heavy-duty vehicles.

Each record represents operational data captured from multiple vehicle sensors.

Target Variable:

| Class | Description |
|---------|-------------|
| 1 | APS-related failure |
| 0 | Non-APS-related failure |
